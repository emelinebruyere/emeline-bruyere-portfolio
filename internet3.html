Internet 3.0

2017

Ce mini-site web aborde la naissance d'internet en adoptant le point de vue
d' une entité virtuelle : Internet 3.0. Cette dernière défend les idéologies des hackers et son accessibilité pour tous dans un univers nocturne de fête foraine. La base de ce projet était la préface du livre « Aux sources de l'utopie numérique », de Fred Turner.

Site réaliser en HTML5, CSS 3.
